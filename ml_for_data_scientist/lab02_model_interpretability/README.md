# Lab 02 — Model interpretability

Two complementary explainability techniques:

- **SHAP** on a Random Forest classifier trained on the Adult Income dataset (`adult.csv`): TreeExplainer values, global feature importance, beeswarm and dependence plots, force plots and textual explanations for three individual predictions.
- **LIME** on a pretrained ResNet50: superpixel-based explanations for five ImageNet-class photos (`*.jpg`), with a discussion of whether the highlighted regions match human intuition.

**Notebook:** `interpretability_shap_lime.ipynb`
