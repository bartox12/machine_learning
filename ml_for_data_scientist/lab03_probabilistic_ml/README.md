# Lab 03 — Probabilistic ML & time series

- `bayesian_regression_gaussian_processes.ipynb` — Bayesian linear regression in **PyMC**: MLE baseline, MCMC sampling (4 chains) with convergence diagnostics, posterior distributions and 94% HDI, posterior predictive checks, Bayes vs MLE comparison; then **Gaussian Process** regression on synthetic 1D functions.
- `time_series_arima.ipynb` — **ARIMA** modelling of Apple stock prices (`Apple_stock_price.csv`): stationarity testing (ADF), ACF/PACF analysis, order selection by AIC, residual diagnostics and forecast validation.
